Welcome, we are excited to have you onboard!

If you are reading this, you are probably one of the first people to get their hands on the Generic Node Sensor Edition. We are extremely excited to have you as a part of our community!

The Generic Node Sensor Edition is a unique device for several reasons:

  • It is a functional device that can be used for many LoRaWAN applications
  • It is a production-ready device that incorporates STMicroelectronics’ new STM32WL SOC
  • It is a reference design hardware
  • It has a source available firmware

Production rollout

We started with shipping 100 devices to developers and community members as part of a beta rollout.

Now that this phase is completed, the Generic Node Sensor Edition is currently being produced at scale. We started with producing and shipping 500 devices and we are ramping up to reach 10,000 devices.

Where to start?

Check out Generic Node Sensor Edition Hardware and Software sections to get familiar with the device.

Using the vanilla application

Each device comes pre-programmed with an example application that lets you experience some of the Generic Node Sensor Edition capabilities and functionalities.

Check out this guide on how to get started using the vanilla application.

Are you a developer?

If you are a developer, please read Hardware Setup and Software Setup sections for instructions on how to install all the necessary tools and prepare yourself for hardware and software development.

Visit Sensor Edition Tutorials page for some great, easy-to-follow video guides. Also, check out the Applications section for many application examples.

Where to contribute?

We use the Generic Node Sensor Edition Github repository in our development. You can file an issue to request a feature or report a bug.

The Generic Node Forum is where we have discussions and where you can provide feedback.

Have fun!

You can try out some cool features that are only possible with the Generic Node Sensor Edition, like:

  • QR code claiming
  • Secure element claiming
  • Adding external sensors
  • Using FreeRTOS with LoRaWAN
  • LoRaWAN FUOTA

We look forward to seeing all the awesome projects built with the Generic Node Sensor Edition! We will share with you all the cool things developed within the community.
