Quick Start

The Generic Node platform is designed to provide an easy way to get started with device development.

To get started with the Generic Node platform, you typically need two things:

  1. Generic Node supported hardware
  2. Generic Node supported software

For example, if you are planning to use the Generic Node Sensor Edition, you would need:

  1. Generic Node Sensor Edition board
  2. Generic Node Sensor Edition application

Get a device

Generic Node Sensor Edition is currently in the production phase. Order one at genericnode.com!

Explore your node’s features

To learn more about your node and set it up for use, follow the hardware and software setup guides contained in your node’s corresponding section:
