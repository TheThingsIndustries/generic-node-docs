Getting a Generic Node

The Generic Node platform is designed to provide an easy way to get started with device development.

Choosing a device

To get started with the Generic Node platform, you would typically need two things:

  1. Generic Node supported hardware
  2. Generic Node supported software

For example, if you are planning to use the Generic Node Sensor Edition, you would need:

  1. Generic Node Sensor Edition board
  2. Generic Node Sensor Edition application

Setting up your node

To learn to setup your Generic Node Sensor Edition, follow these sections:

  • Sensor Edition Hardware Setup Guide
  • Sensor Edition Software Setup Guide

Exploring the node features

If you want to learn more about your node, take a look at the following sections:

  • Hardware
  • Software
  • Sensor Edition
  • Tracking Edition
