STM32CubeIDE

If you are using STM32CubeIDE as your build environment, then using the same tool to program your Generic Node Sensor Edition is convenient.

Steps to reproduce

  1. To program and run or debug Generic Node Sensor Edition, right-click on it and select Run As or Debug As. When programming, make sure you select STM32 Cortex-M C/C++ Application in the Run As or Debug As section.
