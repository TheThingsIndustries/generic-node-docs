Programming Software

Become familiar with programming your executables to your Generic Node Sensor Edition with the ST-LINK programmer.

ST-LINK

The ST-LINK debugger and programmer is required to program the Generic Node Sensor Edition. There are multiple versions of this probe, but most of them have a similar pin-out and functionality. In this guide, we will be using the ST-LINK/V3SET probe. Commonly used ST-LINK probes feature blinking LEDs to indicate the programming process.

Wiring your ST-LINK to the Generic Node Sensor Edition

The wiring from the ST-LINK/V3 programmer → Generic Node Sensor Edition is as follows:

  1. VCC → VCC
  2. GND → GND
  3. CLK → CLK
  4. DIO → SWD
  5. NRST → NRST

Optionally you can connect RX and TX pins to work with UART (either for sending and receiving messages or programming).

  1. RX → TX
  2. TX → RX
Warning:
Be careful not to mix VCC with VIN pin on the Generic Node Sensor Edition. VIN should still be powered separately, either with the pin or the battery connector, as it powers the Generic Node Sensor Edition. VCC is used as target voltage indicator.
